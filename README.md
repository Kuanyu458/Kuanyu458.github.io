# Kent Chen — AI Engineer Portfolio

Dark, responsive portfolio for AI engineering work, research, technical writing, and selected freelance services.

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```

## Deployment

The included GitHub Actions workflow builds the Vite app and deploys `dist/` to GitHub Pages. Use this repository as the public user site `Kuanyu458/Kuanyu458.github.io`, then set **Settings → Pages → Source** to **GitHub Actions**.

## Content ownership note

Taiwan Health MCP is not represented as a personal project. It appears only in a technical article, with explicit attribution to HealthyMind Tech and a clear statement that Kent's contribution is LLM application integration and evaluation design.
